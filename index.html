Yotpo Seeder

By: Daniel Lerman

Routes: Add Client


/yotpo-fetch

* required field

Query Parameters:

*shop: Specifies the shop url to retrive reviews for. Use liquid's {{shop.url}}

/yotpo-fetch?shop=exampleshop.com.

pid: Specifies a specific product ID to retieve reviews for. Use liquid's {{product.id}}

/yotpo-fetch?pid=1234567.

/yotpo-add

* required field

Query Parameters:

*product_id: Specifies a unique product ID to add this review to. Use liquid's {{product.id}}

/yotpo-add?product_id=1234567&...

*product_title: The title of the product. Use liquid's {{product.title}}

/yotpo-add?product_title=title goes here&...

*display_name: The reviewer's name.

/yotpo-add?display_name=John Testerson&...

*email: The email of the reviewer.

/yotpo-add?email=example@website.com&...

*review_content: The content of the review.

/yotpo-add?review_content=I love this product&...

*review_title: The title of the review.

/yotpo-add?review_title=This is my review title&...

*review_score: The review score.

/yotpo-add?review_score=5&...

product_description: The description of the product. Use liquid's {{product.description}}

/yotpo-add?product_description=Made from the finest materials.&...

product_image_url: The url of the product image. Use liquid's {{product.featured_image.src}}

/yotpo-add?product_image_url=http://website.com/image.jpg&...
